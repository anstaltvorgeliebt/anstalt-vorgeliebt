## Bilder ersetzen
Aktuell sind in `assets/img/` SVG-Platzhalter. Ersetze sie durch echte Fotos (JPG/PNG/WebP) und passe die Dateinamen in den HTML-Dateien an, oder behalte die Namen und ersetze die Inhalte.

## Bewertungen 1:1
In `assets/js/main.js` findest du `reviewsData`. Ersetze dort die Platzhaltertexte **1:1** durch eure echten Google-Bewertungen (Text + Name/Ort).
